Starbucks Data Cafe

Starbucks Data Café is a platform that supports Starbucks data workers and consumers, and partners who are data curious. This platform can provide content that enables users to learn about the data at Starbucks with an emphasis on data that is available in the analytics platforms. It allow users to understand how that data is defined, sourced, and transformed.

Starbucks Data Café provides a “data marketplace” where users can learn about dataset availability and access this data based on their business role. It allow users to discover data that can improve their work processes and resulting decision making. This is a platform for the sharing of information and resources across the data analytics community. It also include the ability to collaborate across projects and optimize the knowledge and skills of the Starbucks data analytics community.
